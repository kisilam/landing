Скрипт сокращений ссылок с помощью Twitter.
Работает через API левого аккаунта Twitter, присутствует рандомизация твита, да бы избежать дублей и банов
Требуемые языки и расширения:
PHP 7 - 7.3
Установка до боли простая, залить в корневую папку домена, и перейти по адресу domain.com/index.php
